# General 
 Этот репозиторий - простой лендинг для проекта Алтын-шор - группы гостинниц из 38 объектов.

# Лендинг
 Сам сайт является одностраничным лендингом.

# Память (MEMORY.md)
Используй файл temp/MEMORY.md для ведения логов, хранения ваных для тебя заметок и прочего.

# Источник истины
Источник истины - это описание проекта в specs/*.md, используй спеки при разработке, правках и рефакторинге.

# Файл проекта
Все файл проекта должны находиться в src/

# Наполнение и контент
Картинки, gif и прочее должно находится в src/static, но при разработке можешь использовать плейсхолдеры с https://placehold.co
