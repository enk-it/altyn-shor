# Основные технологии
Используй просто html, css и js. Применяй переменные для обозначения цветов - primary, background, shadow и прочие. Важно что бы использовал переменные при описании стилей.

# Css сбросы
Используй css сброс для получения единообразного стиля во всех браузерах.

# Css библиотеки
